# Style Pack — editorial-premium

Use for markets that need typographic boldness.

Core feel:
- bold typography
- premium editorial composition
- confident spacious hierarchy
- stronger design personality

Recommended:
- base background: black or graphite
- primary accent: local brand color
- oversized but controlled headings
- light contrast cards allowed
- best hero variant: `hero-editorial`
- good steps variant: `steps-timeline`
- good paths variant: `paths-choice-cards`
- good final CTA: `final-cta-dual-card`

Use for:
- pages that need to feel more designed and less dashboard-like
- premium markets where strong type can carry identity
- partnership pages requiring clarity and authority

Avoid:
- using oversized headings without hierarchy
- making body copy too wide
- pairing `hero-editorial` with heavy dashboard components unless approved
