# Style Pack — red-green-national

Use for flag-driven identity pages.

Core feel:
- dark base
- strong national accent behavior
- restrained surfaces to prevent overload

Recommended:
- primary accent: strong red
- secondary accent: green
- background remains dark
- restrained visual components
- limited accent use
- clear hierarchy

Use for:
- markets where national palette is important
- pages that need local identity through color

Avoid:
- giant flags everywhere
- tourism poster look
- visual overload from red + green competition
- relying only on flag colors for identity
